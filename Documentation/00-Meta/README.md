## üéÆ Project Overview

**OCTP** is a top-down, real-time 2D game featuring:
- **Snake-like movement controls** - Novel movement mechanic
- **RPG party-building** - Character roster and party management
- **Real-time combat** - Action-oriented battles
- **Progression systems** - Leveling, unlocks, and equipment

**Platform**: Unity 6000.3.7f1 with Universal Render Pipeline (URP)
**Team Size**: Small team / solo developer
